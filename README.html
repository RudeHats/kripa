What's inside

Core Features

Everything you need to monitor, understand, and act on air quality — built with a clinical-grade AI layer for health-sensitive users.

🌍
Interactive Live Map
Click anywhere on the Leaflet map to fetch real WAQI sensor data for that precise location. Pins update instantly with AQI colour coding.
📊
Real-time AQI Gauge
Animated SVG gauge ring showing the current index alongside 24-hour average, daily peak, and low — refreshed every 60 seconds.
🤖
AeroGuard AI Chatbot
Powered by Groq's Llama 3.3 70B with function-calling tools. Provides clinical air quality assessments, predicts tomorrow's AQI, and tailors advice for asthma/COPD patients.
🏙️
Global City Search
Search any city worldwide. Uses the WAQI search endpoint to resolve city names to exact sensor UIDs before fetching live data.
🧬
Health Bio-Data Context
Upload medical documents (PDF/DOCX) in the Settings panel. The AI reads your health profile and overrides default severity protocols to give you personalised survival guidance.
📷
Camera / Image Analysis
Attach any image or use your live webcam feed. The AI performs a CNN-style analysis pass and delivers environment-quality metrics derived from the scene context.
⚠️
Alert Banners
Auto-surfaced alert banners when AQI crosses critical thresholds. Bell icon pulses red at AQI > 150 and triggers an actionable dismissible warning.
📈
24-Hour Trend & Forecasts
Recharts-powered area chart of the last 24 hours plus a prediction card for tomorrow. Colour tracks the current AQI severity band dynamically.
Reference

AQI Scale

The dashboard uses the standard US EPA AQI scale. Each band has dedicated colour coding, alert logic, and clinical health guidance.

Good (0–50)
Moderate (51–100)
Unhealthy for Sensitive Groups (101–150)
Unhealthy (151–200)
Very Unhealthy (201–300)
Hazardous (301+)

Getting Started

Setup Guide

Up and running in under 5 minutes.

1
Clone & install dependencies
git clone https://github.com/RudeHats/kripa.git
cd kripa
npm install
2
Set up environment variables
# Copy the template and fill in your keys
cp .env.example .env.local   # for Next.js
cp .env.example .env         # for Python backend

# Keys you'll need:
NEXT_PUBLIC_WAQI_TOKEN=your_waqi_token   # https://aqicn.org/api/
GROQ_API_KEY=your_groq_key              # https://console.groq.com/
WAQI_API_TOKEN=your_waqi_token
HF_TOKEN=your_huggingface_token         # optional
3
Install Python dependencies & start the AI backend
pip install fastapi uvicorn groq requests python-dotenv
python Chatbot.py
4
Start the Next.js dev server (new terminal)
npm run dev

Open http://localhost:3000 — the dashboard is live.
